Cognitive Bias

Anchoring effect

The tendency for the first salient number, frame, or option to pull later estimates toward itself even when it is arbitrary or weakly relevant.

Estimation & Prediction Structured process Forecasting & planningPersonal decisions

Default mental move

Start from the first offered value and adjust insufficiently away from it.

What it distorts

It biases negotiation, forecasting, pricing, and planning by shrinking the range of considered outcomes.

Common trigger

Opening offers, first forecasts, suggested budgets, and round numbers.

First countermove

Record an independent estimate before exposing yourself to any outside number.

Mechanism

Initial values set a starting point, and later adjustment is usually too small. The mind mistakes a convenient first reference for an informative one.

Signals You Are In It

  • Different opening numbers produce different 'reasonable' endpoints.
  • People defend estimates they generated only after seeing a suggested figure.
  • The discussion keeps orbiting a first number that no one has justified.

Why This Category Matters

These biases distort numerical judgment, forecasting, calibration, and first-pass estimates.

What anchor, confidence level, or forecast frame is quietly steering the estimate?

Counter It

From detection to intervention

Bias work is less about slapping on a label and more about changing what happens next in the decision process.

Spot It

  • Ask what the estimate would have been if a different opening number had appeared first.
  • Check whether the anchor is actually evidence or merely the first thing mentioned.
  • Look for narrow revisions that never escape the initial frame.

Slow It

  • Generate your own range before seeing the offered anchor.
  • Use outside-view distributions to set a fresh range.
  • Require a written reason for every major adjustment.

Reframe It

  • Treat the first number as noise unless it earns authority.
  • Ask which variables actually determine the answer.
  • Work from reference classes outward rather than from the anchor inward.
