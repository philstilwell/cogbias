Cognitive Bias

Base-rate neglect

The tendency to underweight general prevalence information when vivid case-specific details are available.

Evidence & Sampling Structured process Research & evidenceForecasting & planning

Default mental move

Let the case description crowd out the prior odds.

What it distorts

It makes rare outcomes seem too plausible and ordinary outcomes seem too negligible.

Common trigger

Diagnostic reasoning, legal reasoning, medical testing, and forecast updates with vivid narratives.

First countermove

Write down the outside-view base rate before discussing the specific case details.

Mechanism

Concrete stories feel richer than abstract frequencies, so the mind overuses individuating detail and underuses prior probabilities.

Signals You Are In It

  • People talk richly about the case but cannot state the prior probability.
  • A vivid detail is treated as decisive without checking how common the outcome is overall.
  • Probability judgments drift far from the background rate for weak reasons.

Why This Category Matters

These biases distort what counts as representative evidence and which cases even make it into view.

What base rates, missing cases, or unobserved failures are absent from the sample?

Counter It

From detection to intervention

Bias work is less about slapping on a label and more about changing what happens next in the decision process.

Spot It

  • Ask what the base rate is for the outcome in the relevant reference class.
  • Check whether the case details are actually strong enough to swamp the prior.
  • Notice whether the narrative feels more informative than it really is.

Slow It

  • State the prior explicitly before updating.
  • Use simple likelihood-ratio language if full Bayes is too heavy.
  • Compare the vivid case against several equally vivid alternatives with different base rates.

Reframe It

  • Treat priors as part of the evidence, not as background wallpaper.
  • Ask how surprised you should be by this case if the base rate were true.
  • Remember that unusual stories happen inside ordinary distributions.
